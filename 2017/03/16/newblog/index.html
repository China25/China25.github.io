实验一 在Github上用Markdown创建技术博客

一、实验目的

  1. 熟悉Github网站
  2. 掌握Markdown语法
    二、实验任务
  3. 注册Github账号
  4. 创建Gihub Pages repository
  5. 发布第一篇博客
  6. 学习Markdown
  7. 利用Hexo生成博客并部署到Github
    三、实验步骤与结果
    一、本地配置Hexo
    1安装git
    2安装node
    3设置npm镜像为淘宝
    4 npm config set registry https://registry.npm.taobao.org
    5安装hexo npm install -g hexo-cli
    6初始化站点
    hexo init myhexo
    cd myhexo
    npm install
    7浏览本地站点
    hexo server
    二、在Github上部署hexo
    在Github上新建repository，名称必须是用户名.github.io，比如我的是zhushaojun.github.io
    修改配置文件_config.yml的deploy部分，注意空格
    type: git
    repository: https://github.com/zhushaojun/zhushaojun.github.io.git
    branch: master
    1安装 hexo-deployer-git
    npm install hexo-deployer-git –save
    2配置git全局邮箱与名字
    git config –global user.email “shj.zhu@gmail.com”
    git config –global user.name “Shaojun Zhu”
    3生成博客
    hexo generate
    4部署博客
    hexo deploy
    三、添加新文章
    1生成新文章
    hexo new newblog
    用编辑器修改文章：myhexo\source_posts\newblog.md
    生成并发布到Github
    2
    hexo d –g

四、实验小结
学习创建博客。
